# 🏠 HostAssist AI

HostAssist AI is a simple web application that helps Airbnb hosts automatically generate professional replies to guest messages using AI.

---

## 🚀 Features

* ✉️ Instant AI-generated replies to guest questions
* 🌍 Automatic language detection (Greek & English)
* 🧠 Context-aware responses based on property details
* ⚡ Fast and easy to use interface

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML / CSS
* OpenAI API

---

## 📸 How It Works

1. Enter your property information
2. Type a guest question
3. Click **Generate Reply**
4. Get a professional response instantly

---

## 💻 Installation (Local Setup)

Clone the repository:

```bash
git clone https://github.com/GKodot/HostAssist.git
cd HostAssist
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create an environment variable for your API key:

```bash
export OPENAI_API_KEY=your_api_key_here
```

Run the app:

```bash
python app.py
```

Open in browser:

```
http://127.0.0.1:5000/
```

---

## 🔐 Environment Variables

You must set your OpenAI API key:

```
OPENAI_API_KEY=your_api_key_here
```

⚠️ Never share your API key publicly.
